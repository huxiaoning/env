03 Gitbook安装配置 

$ npm install gitbook-cli -g

# 初始gitbook
$ gitbook init
# 构建站点静态资源 _book/ 目录
$ gitbook build
# 启动一个服调试站点
$ gitbook serve

# 安装插件
$ gitbook install
